# Conway's Game of Life


## Installation

1. Clone this repository on your local file system

    ```
    $ cd /path/to/install/location
    $ git clone git@github.com:martinchu/Game-of-Life.git
    ```

2. Install dependencies

    ```
    $ npm install
    ```

3. Start project

    ```
    $ npm run start
    ```
